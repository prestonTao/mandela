# mandela

#### 介绍
mandela是一个去中心化分布式项目
包含功能：
1.云存储。
2.聊天。
3.web 3.0.
4.域名系统。
5.区块链通证经济。

本系统率先提出web3.0概念，利用云存储，域名，聊天等模块作为web服务器的脚手架，可以快速搭建web服务器，减少中小企业运营成本。
